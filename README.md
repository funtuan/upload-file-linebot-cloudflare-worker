# 超商檔案列印小幫手

「超商檔案列印小幫手」是一個 LINE bot 的應用，可以協助使用者將檔案上傳至超商，取得超商列印編號，並發送給使用者，使其前往 ibon 列印。

## 如何使用

1. 將「超商檔案列印小幫手」加入 LINE 好友，ID:@fua2204f
2. 傳送檔案給「超商檔案列印小幫手」
3. 等待「超商檔案列印小幫手」回覆，取得超商列印編號
4. 前往 ibon 列印，並輸入超商列印編號

## 如何部署

```sh
$ npm run deploy
# or
$ yarn run deploy
# or
$ pnpm run deploy
```
